# Images Downloader
Service for download images from URL.

## Technologies
- .NET 4.6;
- ASP.NET MVC;
- ASP.NET Web API.

## Packages
- Hangfire (1.4.6);
- HtmlAgilityPack (1.4.9);
- Newtonsoft Json (6.0.4).

## UI dependencies
 - bootstrap (3.0.0);
 - JQuery (1.10.2);
 - respond.js (1.2.0).
